# Cryptogram

Логическая игра — шифрограммы на русском и нескольких языках. В оригинале флеш-версия, но здесь переписанный SwiftUI-вариант с подсказками, таймерами и лёгкими интеракциями.

## Фишки
- фразы на русском, английском, немецком, французском, итальянском, испанском, португальском;
- управление клетками и автопереход при правильном вводе;
- режим подсказки, ежедневное пополнение и подсказка за рекламу;
- окно победы с автором и плавный переход к следующей фразе;
- экран последнего шанса, который предлагает посмотреть рекламу перед проигрышем.

## Сборка и запуск
```bash
xcodebuild -project Cryptogram.xcodeproj -scheme Cryptogram -sdk iphonesimulator -configuration Debug build
```
затем можно запустить руками через Xcode или `xcrun simctl`.

Для установки на реальное устройство используйте `ios-deploy --bundle <path>/Cryptogram.app --justlaunch` после сборки с `-destination 'generic/platform=iOS'`.

## Локализация
Тексты хранятся в `AppLocalization.swift`, новые строки добавляются в перечисления языков и словари.

## Поддержка
Если нужно допилить режимы рекламы или добавить новые фразы — правьте `ProgressSupport.swift`, `PhraseStore.swift` и соответствующие `.txt`.
